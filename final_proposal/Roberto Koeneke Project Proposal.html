Project Description

I will conduct the final project as an individual, and will focus on writing a white paper that thoroughly describes what a good web application (or website) localization workflow should look like. The purpose of the project is to familiarize myself with the web app localization process, especially the workflow, and learn to articulate what is necessary and proper for developing a good localization workflow for web apps. As a localization management student, it is highly valuable to be able to communicate the requirements and features of great web app localization to the localization team.

Project Outline

  • meet with Professor Suitter to discuss project goals.
  • conduct research of web app localization workflows.
  • select example localization workflow for web app.
  • write paper on good web app localization workflow.
  • deliver white paper to Professor Suitter via GitHub and iLearn.
